# Open Software Homebrew tap

Homebrew casks for [Open Software](https://opensoftware.co) apps.

## Install June

```sh
brew install --cask open-software-network/tap/june
```

June is an AI assistant for meeting notes, dictation, and agent work on macOS.
Source: [open-software-network/os-june](https://github.com/open-software-network/os-june).

The app updates itself, so `brew upgrade` is rarely needed. The cask version is
bumped automatically by the June release pipeline.
